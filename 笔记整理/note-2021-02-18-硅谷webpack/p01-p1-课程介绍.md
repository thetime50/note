## 03-webpack体验

```cmd
npm init  
npm i webpack webpack-cli -g
npm i webpack webpack-cli -D ::开发依赖
```

webpack 入口文件  
1. 运行指令  
    - 开发环境指令
    - webpack ./src/index.js -o ./build/built.js --mode=development  
        - 打包信息：  
            Hash: 哈希  
            Version: webpack版本  
            Time: 时间  
            Build at: 详细信息  
    - 生产环境指令

```
package.json:
    "scripts": {
        "webpack": "webpack"
    },

npm run webpack -- ./src/index.js -o ./build/built.js --mode=development
```
