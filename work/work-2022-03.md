### 小程序场景值

onLaunch 和 onShow，或wx.getLaunchOptionsSync 中获取上述场景值

```js

  onLaunch(options) {
    console.log('App onLaunch Scene:', options.scene);//options.scene 是 String 类型的 
  },
  onShow(options) {
    console.log('App onShow Scene:', options.scene);
  },
  ```
